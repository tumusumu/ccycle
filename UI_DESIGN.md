# ccycle 碳循112 — UI 设计文档

## 设计理念
参考健康类APP的温和、友好风格。整体感觉轻松愉快，不要生冷硬。
颜色柔和，卡片化布局，圆润边角。让用户每天打开APP觉得舒服。

---

## 一、色彩系统

### 背景色
- **页面主背景：** `#EEF2F7`（非常浅的蓝灰，参考图片的整体背景）
- **卡片背景：** `#FFFFFF`（白色卡片浮在背景上）

### 主色（碳水/核心功能）
- **主色：** `#4A90D9`（蓝色，用于按钮、链接、活跃状态）
- **主色深：** `#357ABD`（按钮hover/pressed状态）

### 营养素专属颜色（环形图用）
- **碳水 Carbs：** `#F5C542`（暖黄色）
- **蛋白质 Protein：** `#E8A0BF`（粉色）
- **脂肪 Fat：** `#A8D5BA`（薄荷绿）
- **饮水 Water：** `#7EC8E3`（天蓝色）

### 状态颜色
- **完成/成功：** `#5CB85C`（绿色）
- **警告/限制食物：** `#E74C3C`（红色，用于水果、白糖等限制食物标注）
- **未完成：** `#CCCCCC`（灰色）

### 文字颜色
- **标题：** `#2C3E50`（深蓝黑）
- **正文：** `#5D6D7E`（中灰蓝）
- **占位文字：** `#AEB6BF`（浅灰）

---

## 二、字体与间距

- **字体：** 系统默认字体（-apple-system, BlinkMacSystemFont）
- **标题（H1）：** 24px, font-weight 700
- **副标题（H2）：** 18px, font-weight 600
- **正文：** 14px, font-weight 400
- **小标注：** 12px, font-weight 400
- **卡片内间距：** padding 16px
- **卡片间距：** margin-bottom 16px
- **卡片圆角：** border-radius 16px
- **按钮圆角：** border-radius 12px（普通按钮）/ 50px（胶囊形按钮）

---

## 三、核心组件设计

### 1. 营养素环形图（参考图片的 Fat/Protein/Carbs 圆环）
- 4个并列的小圆环：碳水、蛋白质、脂肪、饮水
- 每个圆环外圈是对应的营养素颜色（见上面的专属颜色）
- 圆环内部显示：已摄入克数（大字）/ 目标克数（小字，灰色）
- 下方标注营养素名称（中文）
- 布局：4个圆环放在一个白色卡片里，均匀分布

### 2. 食物卡片（今日食谱列表）
- 每一餐一个卡片（早餐、午餐、加餐、晚餐）
- 卡片左侧有一个彩色的餐次标记（圆点或小标签）
- 餐次名称加粗显示
- 食物列表逐项排列，每项显示：食物名称 + 克数
- 每项右边一个勾选框，完成后变绿色+划线
- **限制食物（水果、白糖、白面）用红色文字 + 红色警告角标注**

### 3. 今日碳水类型标签
- 显示今天是哪种碳水日（低碳日/中碳日/高碳日）
- 用一个圆形标签，颜色对应：
  - 低碳日：`#A8D5BA`（薄荷绿）
  - 中碳日：`#F5C542`（暖黄）
  - 高碳日：`#E74C3C`（红色）

### 4. 饮水进度条
- 横条形进度条，蓝色 `#7EC8E3`
- 显示：已喝 Xg / 目标 4000ml
- 可以点击"+"快速添加

### 5. 底部导航栏
- 4个标签：首页、食谱、运动、我
- 图标 + 中文标签
- 活跃标签用主色 `#4A90D9`，非活跃用灰色
- 白色背景，上方有一条很浅的分割线

### 6. 按钮
- **主按钮（如"下一步"、"开始计划"）：** 主色背景 `#4A90D9`，白色文字，圆角 12px，宽度拉满
- **次级按钮：** 白色背景，主色边框，主色文字
- **胶囊形小按钮（如"+"添加）：** 圆角 50px，较小尺寸

### 7. 卡片样式
- 白色背景
- 圆角 16px
- 淡色阴影：`box-shadow: 0 2px 8px rgba(0,0,0,0.06)`
- padding 16px
